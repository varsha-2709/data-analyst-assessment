# Sales Data Analysis

## About

This was a data analyst assignment where I worked with customer, orders, and shipping data from different sources.

## What I did

* Loaded and checked the data for issues (nulls, duplicates, etc.)
* Wrote SQL queries to answer business questions
* Created basic insights from the data

## Key outputs

* Spending for pending deliveries by country
* Customer-level sales summary
* Most purchased items by country and age group
* Country with lowest sales
* Shipping status breakdown

## Tools used

Google Colab

## Note

The dataset did not have quantity or product ID, so I used available fields like `Item` and `Amount`.
